# 🚀 Java Servlet Programs

This repository contains my practice programs on **Java Servlets**, covering core concepts of web development such as request handling, session management, and form data processing.

---

## 📂 Topics Covered

### 🔹 Servlet Basics

* Creating servlet using Servlet Interface
* GenericServlet class
* HttpServlet class

### 🔹 Servlet Implementation

* Handling client requests (GET & POST)
* Writing response using PrintWriter

### 🔹 Session Management

* HTTP Session tracking
* Using HttpSession

### 🔹 Form Handling

* Handling form data
* Hidden form fields

### 🔹 Server Configuration

* Apache Tomcat server setup
* Deployment and configuration

---

## 💡 Example Programs

* Servlet using Servlet Interface
* Servlet using GenericServlet
* Servlet using HttpServlet
* Handling user input from forms
* Session tracking using HttpSession
* Hidden form field example
